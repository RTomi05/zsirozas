# Zsirozas


Kiinduló helyzet:
Van egy űrlap hány játékos módban játszunk

Kezdődik a játék
Random kisorsolunk mindenkinek 4 lapot

Random sorsolt játékos kezd

Kikér egy lapot: Kattint a négy kártya közül valamelyikre
A többiek eldöntik mit raknak
Ha valaki üti: elviszi az össszes kártyát:
Lesz egy kis panel ahol az elvitt kártyák és az ütések számát
Valamint a "zsírt" érő lapot számát

A játék addig megy amíg el nem fogy a pakli

Ha elfogy a kártya véget ér a játék eldől, hogy ki a nyertes
